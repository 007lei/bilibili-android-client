# OhMyBiliBili
------
莆田系高仿真哔哩哔哩动画客户端

## 简介

高仿B站安卓客户端

##使用说明
该项目所使用Api均来自:
https://github.com/fython/BilibiliAPIDocs

因为B站Api需要Appkey才可正常使用,具体Appkey请自行获取.
放置到Secret的Appkey中即可.
OhMyBiliBili/app/src/main/java/com/hotbitmapgg/ohmybilibili/config/Secret.java
